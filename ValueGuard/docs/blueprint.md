# **App Name**: ValueGuard

## Core Features:

- Incident Reporting: Allow users to anonymously report incidents at their companies that contradict their personal or societal values.
- Incident Categorization: Categorize incidents based on the type of value contradiction (e.g., environmental, social, governance). An AI tool suggests the top 3 categories for each incident reported by the user.
- Company Profiles: Create company profiles aggregating reported incidents. Users can flag any data discrepancies in any company profile.
- Value Search: Enable users to search for companies based on their values and see related incidents.
- User Reviews: Users can submit an edit/review request with new insights for existing incidents in the app.

## Style Guidelines:

- Primary color: Deep teal (#468A8F), a hue known for signaling ethics.
- Background color: Light teal (#E0F2F1), close to the primary, but desaturated to 20% for softer feel.
- Accent color: Dusty rose (#A2505D), an analogous color that will draw the user's attention, creating contrast.
- Body and headline font: 'Inter', a grotesque-style sans-serif with a modern look, suitable for all text.
- Use simple, universally recognizable icons to represent different value categories.
- Employ a clean, card-based layout for displaying incidents and company profiles to facilitate readability.
- Use subtle transitions and animations to provide feedback and enhance the user experience without being distracting.
